  • Author: Naughty Ogre.
  • Premalink: http://www.blendswap.com/blends/view/73457.
  • For Blender 2.52 and up.
  • Description:

    a set of lego-like bricks (no lego logo on top).the set is made as highly cus tomizable via arrays. the set contains basics 1x1, 1x2 and 2x2, plus arrays that allows you to have 1xn, 2xn and nxn bricks, both high and low bricks (nxn brick s are only low).

    for further explainations pleas refer to this tutorial http://youtu.be/46GrWz99Oec where i ex plain how i created the models and how to use them

    the set also contains some basic bricks materials for cycles
